# A/B Testing Analysis (Python)

## Beschreibung
In diesem Projekt habe ich einen A/B-Test ausgewertet, um zwei Varianten miteinander zu vergleichen und zu prüfen, ob ein Unterschied statistisch signifikant ist. Ziel war es, datenbasierte Entscheidungen auf Grundlage von Nutzerverhalten zu treffen.

## Technologien
- Python
- Pandas
- NumPy
- SciPy

## Datengrundlage
Das Projekt basiert auf einem Datensatz mit zwei Gruppen:
- Kontrollgruppe (A)
- Testgruppe (B)

Für beide Gruppen wurden Kennzahlen wie Conversion-Raten und Nutzerverhalten analysiert.

## Vorgehen
Die Analyse erfolgte in mehreren Schritten:

1. Datenbereinigung und Vorbereitung  
2. Berechnung der Conversion-Raten beider Gruppen  
3. Formulierung der Hypothesen:
   - Nullhypothese (H0): Kein Unterschied zwischen den Gruppen  
   - Alternativhypothese (H1): Es besteht ein Unterschied  
4. Durchführung eines statistischen Tests (z. B. z-Test)  
5. Berechnung des p-Werts und Interpretation der Ergebnisse  

## Analyse
Es wurde geprüft, ob die Unterschiede zwischen den Gruppen zufällig entstanden sind oder statistisch signifikant sind. Dabei wurde ein Signifikanzniveau von 5 % verwendet.

## Ergebnis
Die Analyse zeigt, ob die Testvariante eine signifikante Verbesserung gegenüber der Kontrollgruppe darstellt und ob eine Änderung empfohlen werden kann.

## Nutzung
Die Analyse wurde in einem Jupyter Notebook durchgeführt.

Zur Ausführung:
1. Notebook öffnen  
2. Datensatz im gleichen Verzeichnis bereitstellen  
3. Zellen nacheinander ausführen
